
## 数据可视化结果

### 可视化建议

1. **line** - 销售额随时间变化趋势
   - X轴: 销售日期
   - Y轴: 销售额
   - 描述: 展示销售额随时间（日期）的变化趋势，可用于识别销售高峰和低谷。

2. **bar** - 各产品类别总销售额对比
   - X轴: 产品类别
   - Y轴: 销售额
   - 描述: 比较不同产品类别的总销售额，识别最畅销的类别。

3. **scatter** - 销量与利润关系
   - X轴: 销量
   - Y轴: 利润
   - 描述: 探索销量与利润之间的相关性，识别高销量高利润或低利润的异常点。

4. **boxplot** - 不同地区利润率分布
   - X轴: 地区
   - Y轴: 利润率
   - 描述: 比较不同地区的利润率分布情况，查看地区间的差异和异常值。

5. **histogram** - 客户评分分布
   - X轴: 客户评分
   - Y轴: 频率
   - 描述: 展示客户评分的分布情况，了解客户满意度的大致范围。

6. **bar** - 促销活动对平均销售额的影响
   - X轴: 促销活动
   - Y轴: 销售额
   - 描述: 比较有促销和无促销情况下的平均销售额，评估促销效果。

7. **line** - 一周内每日平均销售额
   - X轴: 销售日期_星期名称
   - Y轴: 销售额
   - 描述: 展示一周中每天的平均销售额，识别周末和工作日的销售模式差异。

9. **bar** - 各产品名称总利润对比
   - X轴: 产品名称
   - Y轴: 利润
   - 描述: 比较不同产品名称的总利润，识别利润贡献最高的产品。
